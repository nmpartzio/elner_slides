NER and applications

  • What is NER?
    • Detection and classification of different categories of proper names
  • Applications
    • NER can be used for content recommendation systems (Netflix, newspaper articles).
    • A NER system could also be used to process customer feedback.
    • A NER system can assist in social media content analysis, such as natural disaster localisation services.
    • Research papers and academic articles are another field where NER can find application.
    • NER can help to categorize each news article in defined hierarchies, thus helping to better organize and retrieve content (people, organizations or countries mentioned in them).

Motivation and Contribution

  • Inspired from "Addition of Greek language to spaCy" for Google Summer of Code 2018
    1. Attempt to provide more linguistic resources for the Greek language, focusing on the creation of a Greek NER dataset
    2. Dataset evaluated with five state-of-the-art Deep Neural Network (DNN) models

Learning methods for NER

  • Feature-based supervised machine learning
    • Recent years, deep neural networks (DNN)
  • Unsupervised machine learning
  • Rule-based learning

Corpus pre-processing

  • Preprocessing with spaCy
    • sentence segmentation
    • sentence tokenisation
  • Prodigy was used to annotate the documents.

Annotation specifications

  • BBN TECHNOLOGIES tag-set
    • 29 category types in total
    • 18 selected for the elNER-18 dataset
    • 4 selected (via merging similar categories) for the elNER-4 dataset
  • Used boundary tagging scheme: Inside-Outside-Beginning-End-Single (IOBES)
  • Standard dataset split: train (80%), development (10%), and test (10%)

Evaluation metrics

Precision (P), recall (R), and micro F1-score were computed as performance metrics
\begin{equation} F_i =2 \times \frac{P_i \times R_i}{P_i+R_i} \quad \text { F-macro }= \frac{\sum_i{F_i}}{C} . \end{equation} \begin{equation} P = \frac{ \sum_i{TP_i} } { \sum_i{(TP_i+FP_i)} } \quad R=\frac{ \sum_i{TP_i} } { \sum_i{(TP_i+FN_i)} } \quad \text { F-micro }=2 \times \frac{P \times R}{P+R} . \end{equation}

Representations

  • Distributed
    • Word2Vec
      • Continuous bag-of-words (CBOW) model
      • Skipgram (SG) model
    • Fasttext (use of subword embeddings
      Words are represented by the sum of the n-gram vectors).
    • GloVe (Global Vector model).
      Co-occurrence count matrix, factorization, to get lower dimensional representations.
  • Contextualized
    • ELMo, GPT2, GPT3, BERT
      A word vector can take different values depending on its context.

* elNER: word vectors trained with fastText on Common Crawl and Wikipedia.

Typically, word embeddings are pre-trained by optimizing an auxiliary objective in a large unlabeled corpus, such as predicting a word based on its context.

DNN Models

The criterion for selecting the architectures was their performance in English.

  1. Bidirectional LSTM-CRF
  2. Bidirectional LSTM-GRU
  3. Bidirectional LSTM-CNNs-CRF
  4. Bidirectional LSTM-CNN
  5. spaCy NER model

Word representations and character-based representations was used to capture morphological and orthographic information.

In order to reduce overfitting, dropout is applied after the output of each LSTM layer.

In order to prevent the learning system from depending too heavily on one representation class, dropout is used.

Discussion

  • Best results are obtained from the neural networks who use in last layer a CRF decoding function.
  • Entities with low number of samples and high ambiguity i.e., WORK OF ART, FAC, PRODUCT, EVENT, LAW have worse precision, recall and F-score metrics.

Error Analysis

  1. Inability to correctly address certain acronyms
  2. Abbreviations like "κ.|mr." or "δρ.|dr."
  3. legislation typology like "ν. 1243/2020|law1243/2020"

Issues encountered from erroneous sentence segmentation

Future Directions

  • Annotation strategy will be validated by a second annotator
  • Sentence segmentation improvements
  • Hyperparameter fine-tuning
  • More experiments, including different annotation schemata for word embeddings, such as GloVe, own-trained ELMo, and BERT
